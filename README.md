# fbcmd4j

Cliente de Facebook en línea de comandos

## Instalación

**Clonar el repositorio**  
```
git clone https://github.com/padacal/fbcmd4j.git
```

**Importar a Eclipse**  
1.`File -> Import -> Existing Projects into Workspace`

**Ejecutar .jar**  
1. Abrir la carpeta `bin`.
2. Abrir la terminal con privilegios de administrador en Windows
3. En `fbcmd4j/bin` se encuentra el archivo `fbcmd4j.jar`.
4. `java -jar fbcmd4j.jar`.


## Uso

**Configurar la cuenta de Facebook**   
1. Seleccionar la opción 0 `Configurar Cliente`.
2. Escribir el código que generó la aplicación.
3. Dar permisos a la aplicación.

**Obtener el NewsFeed**   
1. Seleccionar la opción 1 `NewsFeed`.
2. Escribir 'Si'. 

**Obtener el Wall**   
1. Opción 2.
2. Escribir 'Si'.

**Publicar un Estado**   
1. Opción 3.
2. Escribir el estado.

**Publicar un Link**   
1. Opción 4.
2. Escribir el link deseado.
